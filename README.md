# Zen - Weather-Inspired Music & Study Assistant

**AUTHORS: Sean Creveling, Anthony Dayoub, Mael Tshiyonga**

## Project Overview

This is a full-stack web application that combines real-time weather data with intelligent music recommendations and an AI-powered study assistant. The application provides a personalized experience where users can discover music that matches their current weather conditions and mood, while also receiving study tips and music recommendations through an interactive AI chatbot.

## Key Features

- **Weather-Responsive Interface**: The application displays current weather conditions based on the user's location, with a dynamic background that changes based on weather and time of day.

- **AI Study Assistant (Zen)**: An intelligent chatbot that helps users with:
  - Study strategies and tips
  - Music recommendations based on preferences, genres, or mood
  - Personalized music discovery through natural conversation

- **Music Discovery**: Integration with Last.fm API to provide:
  - Genre-based music recommendations
  - Artist and track information
  - Popular chart tracks
  - Similar artist suggestions

- **User Profiles**: Personalized user accounts that store:
  - User preferences and location
  - Visit history and timestamps

- **Modern Tech Stack**:
  - **Frontend**: React with Vite
  - **Backend**: Flask (Python)
  - **Database**: SQLite
  - **APIs**: WeatherAPI, Last.fm, OpenRouter AI, Google Maps
